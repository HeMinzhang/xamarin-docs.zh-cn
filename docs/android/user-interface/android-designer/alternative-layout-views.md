### <a name="conflict-persistence"></a>冲突持久性

冲突文件中保留布局作为 XML 注释，如下所示：

```xml
<!-- Widget Inserted Conflict | id:__root__ | @+id/checkBox1 -->
```

因此，当一个项目是关闭并重新打开，所有冲突仍在那里&ndash;甚至是那些已忽略。
