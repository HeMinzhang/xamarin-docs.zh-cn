内容的管道通常用于在游戏开发优化内容，并设置其格式以便它可以在加载某些硬件上或与某些游戏开发框架。

本文提供了对内容的管道，主要着重于 MonoGame 内容管道，这是与 CocosSharp 和 MonoGame 一起使用的内容的管道实现概念的理解。
