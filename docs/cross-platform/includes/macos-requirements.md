|**Xamarin.Mac**|macOS High Sierra (10.13) SDK|MacOS High Sierra (10.13) SDK 附带[Xcode 9](https://developer.apple.com/library/archive/releasenotes/DeveloperTools/RN-Xcode/Chapters/Introduction.html#//apple_ref/doc/uid/TP40001051-CH1-SW876)和生成使用最新的 macOS Api 的应用程序时需要。|
