# <a name="pcl-reflection-api"></a>PCL 反射 API

Microsoft 开发了一个新的反射 API 可移植类库中使用。 如果你有想要移动到一个 PCL 一些现有反射代码，可能无法工作。 更多信息：

- [不断演变的反射 API](http://blogs.msdn.com/b/dotnet/archive/2012/08/28/evolving-the-reflection-api.aspx)
