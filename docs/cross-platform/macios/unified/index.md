# <a name="unified-api-for-xamarinios-and-xamarinmac"></a>Xamarin.iOS 和 Xamarin.Mac 统一的 API

Xamarin 的统一 API 使可以 Mac 和 iOS 和支持 32 和 64 位应用程序使用相同的二进制文件之间共享代码。 默认情况下，在新的 Xamarin.iOS 和 Xamarin.Mac 项目中使用统一的 API。
