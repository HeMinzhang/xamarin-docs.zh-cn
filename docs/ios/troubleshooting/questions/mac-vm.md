# <a name="can-i-use-a-mac-vm-with-xamarin"></a>可以通过 Xamarin 使用 Mac VM？ 

## <a name="mac-hardware"></a>Mac 硬件
否。请务必按照 Apple 的使用条款。

## <a name="non-mac-hardware"></a>非 Mac 硬件
否;使用 Apple 的条款不允许这种用法。
