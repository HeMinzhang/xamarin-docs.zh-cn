## <a name="core-ml-2-natural-language-framework"></a>核心 ML 2： 自然语言框架

该自然语言框架，属于核心 ML 2，使应用程序可以执行各种类型的语言分析。 例如，它可用来识别词类并确定由文本块的语言。
